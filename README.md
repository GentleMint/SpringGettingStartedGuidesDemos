# SpringGettingStartedGuidesDemos

01_BuildingJavaProjectsWithGradle
===

Reference
---
https://spring.io/guides/gs/gradle/

Usage 1: Run as JVM application with gradle script
---
1. $ git clone *.git
2. $ cd 01_BuildingJavaProjectsWithGradle
3. $ gradlew run

Usage 2: Import into Eclipse IDE
---
1. $ git clone *.git
2. $ cd 01_BuildingJavaProjectsWithGradle
3. $ gradlew eclipse
4. Import the project into eclipse.
